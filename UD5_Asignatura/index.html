BIENVENIDOS A UTAD

LENGUAJE DE MARCAS, CREA TU PROPIA WEB

¿Que aprenderemos de esta maravillosa asignatura, y su no menos maravilloso profesor?

Que aprenderemos en esta asignatura? En esta asignatura aprenderemos a crear desde archivos HTML, hasta JAVASCRIPT, pasando por hojas estilo CSS, implementacion de boostrap, tailwind,¡ y mucho mas!
En esta asignatura aprenderas a programar por ti mismo tu propia pagina web, los objetivos son que seas capaz de desarrollar de manera autonoma, pasando por un gran periodo de aprendizaje dinamico.

UNIDAD 1

LENGUAJES DE MARCAS PARA VISUALIZACION DE INFORMACION

Se analizarán conceptos fundamentales como la estructura de un documento HTML, las etiquetas básicas y su uso, y la sintaxis del lenguaje.

UNIDAD 2

HOJAS ESTILO CSS

Se abordarán temas como el diseño de layouts, la manipulación de colores y fuentes, el uso de selectores, y la creación de reglas de estilo.

UNIDAD 3

DISEÑO WEB RESPONSIVO

Se analizarán las técnicas y buenas prácticas para crear diseños adaptables, como el uso de media queries, También se verá cómo optimizar la carga de imágenes y textos para mejorar la experiencia del usuario en dispositivos móviles.

UNIDAD 4

MAQUETACION WEB: BOOSTRAP Y TAILWIND

maquetación web utilizando frameworks como Bootstrap y Tailwind. Se usaran técnicas para desarrollar aplicaciones web interactivas y dinámicas.

UNIDAD 5

DESARROLLO WEB CLIENTE: JAVA SCRIPT

Adquiere los fundamentos del desarrollo web cliente con JavaScript. Aprende a manipular el DOM, crear interacciones dinámicas y gestionar eventos en una página web.
